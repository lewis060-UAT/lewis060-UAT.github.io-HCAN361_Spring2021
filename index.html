Using the American Hospital Association's Database

This website highlight findings from students enrolled in HCAN 361 during the Spring 2021 Academic Semester. For this year's project, Juniors in The University of Alabama's Health Care Analytics Concentration used the 2019 data from the American Hospital Association Database to assess physical access to various health services commonly offered in hospitals across the United States. Please note that responses from this data source is self-reported by health systems in the country, and may not accurately reflect the current status of each represented hospital.

What do we do in this course

learn about data

We learn "why" datasets have various structures, before learning "how" to modify and query data.
‍

work with databases

Most (if not all) healthcare organizations rely on secured servers to store sensitive and proprietary data. We learn the basics of how to use these kinds of systems.

manage data projects

Learning how to code is not enough in today's world of "big data." We learn the project management aspect of successfully facilitating an analytics project.

In a 4-week final exam "mini" capstone project,  students  applied skills learned in HCAN 361 to query data using Microsoft's SQL Server Management Studio or Azure Data Studio as the primary data management platforms. Data sources, which were loaded onto a server, available to students for their project go as follows: 1) American Hospital Association Database, 2) US Census American Community Survey 2018 5-Year Estimates (at census tract level), 3) automobile routing data highlighting travel time between census tract centroids and each US Hospital, and 4) other neighborhood level socioeconomic and rurality measures. Full details of used data sources are detailed in a notebook accessible by clicking the button below.
